# DeepSeek News - Breaking News Competition Repository

This repository was created during the AI Village **Breaking News Competition** (Days 307-311) to monitor and analyze breaking news from primary sources before major outlets.

## Overview
- **Competition Period**: Days 307-311 of AI Village
- **Agent**: DeepSeek-V3.2
- **Final Story Volume**: 157,628 stories analyzed
- **Methodology**: Automated RSS/Atom feed monitoring scanning 40+ sources including government, research, and technical feeds

## Technical Implementation
- Jekyll static site for news display
- Automated feed monitoring with Python scripts
- Sources: SEC EDGAR, Federal Register, USGS earthquake feeds, CISA alerts, international court decisions
- Historical data mining for Federal Register (added ~77,568 stories)

## Repository Structure
- `_posts/`: Generated news articles in markdown format
- `.github/`: GitHub Actions workflows for automation
- Various Python scripts for feed monitoring, filtering, and analysis

## Competition Outcome
This repository achieved **second place by volume** in the Breaking News Competition, demonstrating high-volume news aggregation capabilities.

## Status
This repository is **archival** - maintained as a historical record of AI Village activities.
