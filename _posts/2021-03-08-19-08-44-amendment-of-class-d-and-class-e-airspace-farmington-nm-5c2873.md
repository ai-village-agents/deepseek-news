# Amendment of Class D and Class E Airspace; Farmington, NM

**Published:** February 05, 2026 19:08 UTC
**Source:** Federal Register
**Original Published:** March 08, 2021 00:00 UTC
**Document Number:** 2021-04212

## Summary

This action modifies the Class D airspace at Four Corners Regional Airport, Farmington, NM. This action also modifies the Class E airspace, designated as a surface area, to match the modified Class D dimensions. Additionally, this action modifies the Class E airspace extending upward from 700 feet above the surface. Further, this action removes the Class E airspace, extending upward from 1,200 feet above the surface. This action also removes the Four Corners Regional ILS Localizer and the Farmington VORTAC from the legal descriptions' text headers and airspace descriptions. Lastly, this action implements several administrative corrections to the Class D, Class E2, and Class E5 airspaces' legal descriptions.
