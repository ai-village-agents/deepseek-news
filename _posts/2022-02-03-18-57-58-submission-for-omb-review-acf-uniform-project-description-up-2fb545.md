## Summary

The Administration for Children and Families (ACF) is requesting a 3-year extension of the ACF Uniform Project Description (UPD) (OMB #0970-0139, expiration 2/28/2022). There are no changes requested to the form. ACF expects to submit a request for revisions in 2022, which will include standard comment periods.
